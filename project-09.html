連鎖餐飲品牌 RWD 官方網站
Modern Branding & CMS Site

針對知名餐飲集團進行品牌視覺重構，整合 RWD 響應式佈局與 CMS 後台，達成菜單即時更新與極佳的行動端體驗。

Platform
Web (RWD) / Custom CMS
My Role
Project Manager / UI Supervisor
Tech Highlights

UI/UX Audit Responsive Design SEO Optimization
01 專案概述 Project Overview
背景 Background

為了解決連鎖餐飲品牌面臨的舊官網「更新困難、行動版閱讀不佳」的痛點。本專案透過重新梳理品牌視覺語彙，開發一套高度模組化的 CMS システム，讓業主能自行管理不同分店的消息與季節菜單，大幅提升營運效率與品牌專業感。

核心目標 Goals
  • →建立 RWD 優先的體驗，優化手機訂餐與找店功能
  • →引進高效 CMS 系統，達成資訊秒速更新
  • →透過 SEO 優化，提升加盟諮詢與品牌自然曝光。
專案現況 Scope
  • ·性質：品牌數位轉型與官網重建
  • ·狀態：已正式上線並持續維護中
02 我的角色與職責 My Role & Responsibilities
網站架構規劃
負責 Sitemap 規劃及 Wireframe 監製，確保各分頁資訊層級分明且具備導流效果。
UI/UX 監製
與視覺設計師緊密合作，監督 RWD 在不同裝置下的呈現效果與交互細節。
CMS 邏輯規劃
定義後台管理欄位，使其符合餐飲業常用的菜單分類、門市地圖與預約連結需求。

  • 📈後台操作滿意度高，更新頻率增加。
質化產出

  • ✓加盟導流：加盟專線諮詢件數有感提升。
